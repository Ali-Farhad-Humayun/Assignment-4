Difference Between HTML and HTML5

HTML HTML5
No semantic elements like <header>, <footer>, etc. Introduces semantic elements like <header>, <footer>, <article>, etc.
Relies on third-party plugins (e.g., Flash) for multimedia. Supports native multimedia with <audio> and <video> tags.
Limited form input types (e.g., text, password). Introduces new input types like date, email, url, range, etc.
